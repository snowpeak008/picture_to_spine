# M03 项目、导入、母版与审批原子计划

M03 只接收用户主动选择的本地 PNG/JPEG/WebP 图片和文本，不调用公网生图 API。图片先暂存、preflight、人工确认，再进入 CAS；母版 candidate、approved revision 与 StyleSpec 均不可变。审批、ack、waiver 是不同命令，AI 与测试永不代签。

M03 的全部执行证据必须严格使用 F2S-DEV-M02-001 拥有的 `schemas/src/evidence.schema.json` 唯一 `EvidenceEnvelope`；任务专属内容只写 `payload`，禁止平行顶层字段或别名。

## F2S-DEV-M03-001 — 项目创建与打开

#### 1. 任务头

- 目标：创建可搬移项目目录、打开并恢复现有项目、维护不含敏感内容的最近项目引用。
- 非目标：不自动迁移损坏项目，不把 redb 当源，不扫描用户磁盘寻找项目。
- 预计：3d。

#### 2. 追踪

- FR/NFR：F2S-FR-PROJ-001、F2S-FR-PROJ-002；ADR：F2S-ADR-STO-001。
- 风险：路径越界、同目录双写者、最近项目泄漏、打开旧/损坏项目时部分修改。
- 上游：F2S-DEV-M02-003、F2S-DEV-M02-005、F2S-DEV-M02-006、F2S-DEV-M02-007、F2S-DEV-M02-008；下游：F2S-DEV-M03-002、F2S-DEV-M03-003、F2S-DEV-M03-004、F2S-DEV-M03-005、F2S-DEV-M03-006。
- 携带：P0/P1/P2 无。

#### 3. 输入

- ProjectStore/Command/Recovery ports；project schema v1。
- config：用户显式选择项目根；fixture：中文/空格/长路径、只读、占用、损坏 manifest。

#### 4. 输出

- crates/domain/src/project/project_identity.rs、project_manifest.rs：领域值，owner 本任务。
- crates/application/src/project/create_project.rs、open_project.rs、recent_projects.rs：use case，owner 本任务。
- tests/project/create_open.rs、open_failures.rs：测试，owner 本任务。
- 再生性：领域/Application 源和测试为受版本控制源；recent-project 索引可重建，项目 manifest 不可覆盖。

#### 5. 设计边界

- UI 只提供用户选择 token；Application 验证路径并调用 ProjectStore；Adapter 执行 IO。
- 使用 Factory、Repository、Unit of Work；拒绝全盘扫描、raw path shell、打开即写迁移、最近列表存图片/prompt。

## F2S-DEV-M03-002 — 图片导入

#### 1. 任务头

- 目标：冻结可信导入边界，暂存用户选择的 PNG/JPEG/WebP，生成 SourceArtifact provenance，并在 preflight 成功后原子提升 CAS。
- 非目标：不信任扩展名，不接收 URL/API，不在 preflight 前完整解码或提升 CAS，不保留失败图片。
- 预计：3d。

#### 2. 追踪

- FR/NFR：F2S-FR-IMP-001、F2S-FR-IMP-003；ADR：F2S-ADR-INPUT-001。
- 风险：解压炸弹、超大文件 OOM、TOCTOU、magic/扩展伪装。
- 上游：F2S-DEV-M03-001、F2S-DEV-M02-006；下游：F2S-DEV-M03-003、F2S-DEV-M03-004、F2S-DEV-M03-006。
- 携带：P2 F2S-R3A-B10-P2-001；P0/P1 无。

#### 3. 输入

- SourceArtifact/import schema、CAS/ProjectStore、用户选择的只读 file token。
- 冻结 inclusive 限值如下；配置只能在 1..默认值内收紧，不能提高默认值。编译期绝对硬限始终第二次校验，损坏/迁移/调试配置高于默认值时拒绝启动而非 clamp。

| 指标 | 默认上限（含） | 绝对硬限（含） |
| --- | ---: | ---: |
| encodedFileBytes | 67,108,864（64 MiB） | 268,435,456（256 MiB） |
| width、height 各自 | 8,192 | 16,384 |
| decodedPixels=width×height | 67,108,864 | 134,217,728 |
| decodedRgbaBytes=decodedPixels×4 | 268,435,456（256 MiB） | 536,870,912（512 MiB） |
| expansionRatioLimit | 100 | 200 |

- V1 只接受 magic 确认的 8-bit PNG/JPEG/WebP；encodedFileBytes 必须大于 0。比例用无浮点 checked-u128 判定 decodedRgbaBytes ≤ encodedFileBytes×limit；任何乘法溢出先拒绝。

#### 4. 输出

- crates/domain/src/import/import_limits.rs、source_artifact.rs 与 crates/application/src/ports/import_preflight.rs：限值、provenance 和 preflight port，owner 本任务。
- crates/application/src/import/stage_image.rs、import_image.rs：暂存/提升事务，owner 本任务。
- tests/import/image_import.rs：导入测试，owner 本任务。
- 再生性：限值、port、use case 和测试为受版本控制源；staging 可清理重建，SourceArtifact/CAS 不可原位重生。

#### 5. 设计边界

- Domain 冻结限值/SourceArtifact；Application 编排 bounded reader、preflight port、CAS；文件 Adapter 由 M02 提供。
- 使用 Staging Area、TOCTOU-resistant hash、Unit of Work；拒绝 extension trust、read-all-before-limit、URL import、可配置放宽。

- done 不等于所有真实解码器 VERIFIED；真实大文件/OOM 矩阵未跑时 UNVERIFIED。

## F2S-DEV-M03-004 — 母版与StyleSpec
